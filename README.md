# Skull3D

A minimal 3D web scene built with **React** and **Three.js (React Three Fiber)**.  
Features multiple interactive 3D models including a cat skull, AR15 rifle, and more - with a personal signature — @kannssai.

---

## Features

- Multiple 3D Models: Switch between different models using the dropdown selector
- Interactive Controls: Rotate, zoom, and explore models with smooth camera controls  
- Lock/Unlock System: Toggle between guided view (locked) and free exploration (unlocked)
- Smart Loading: Animated loader for smooth model transitions
- Camera Reset: Instant reset to optimal viewing position
- Mobile Optimized: Responsive design with Instagram browser compatibility
- Optimized for mobile and desktop
